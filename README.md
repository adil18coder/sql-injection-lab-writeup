# SQL Injection Lab – TryHackMe Writeup

This repository contains my walkthrough and notes for the SQL Injection Lab room on TryHackMe.

The purpose of this lab is to demonstrate how SQL injection vulnerabilities occur and how attackers can exploit them to bypass authentication and extract data from a database.

## Skills Practiced

- SQL Injection fundamentals
- Authentication bypass
- Error-based SQL injection
- UNION based SQL injection
- Web parameter manipulation

## Tools Used

- Web Browser
- Burp Suite (optional)
- SQLMap (optional)
- Manual SQL payloads

## Techniques Used

- Boolean based injection
- Authentication bypass
- UNION SELECT data extraction

## Lab Objectives

1. Detect SQL injection vulnerabilities
2. Bypass authentication
3. Enumerate database columns
4. Extract data using UNION queries

## Disclaimer

This writeup was created for educational purposes and all attacks were performed inside the TryHackMe lab environment.
